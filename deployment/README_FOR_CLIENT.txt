INVENTORY MANAGEMENT SYSTEM - PRELIMINARY VERSION
===========================================

IMPORTANT NOTE: This is a preliminary version of the software. Updates with additional features and bug fixes will be provided shortly.

Prerequisites:
-------------
1. Java Runtime Environment (JRE) 24
   Download from: https://www.oracle.com/java/technologies/downloads/#jdk24-windows
   
2. MySQL Server 8.0 (Included in the package)

Installation Steps:
-----------------
1. Extract this ZIP file to a location on your computer (e.g., C:\InventorySystem)

2. First-Time Database Setup:
   a. Double-click 'start_db.bat' to start the MySQL server
   b. Wait for about 30 seconds for the server to initialize

3. Running the Application:
   a. Make sure the database is running (step 2)
   b. Double-click 'run_app.bat' to start the application
   c. Login with your provided credentials

Default Login Credentials:
------------------------
Username: admin
Password: admin123

Known Limitations (To be addressed in upcoming updates):
----------------------------------------------------
- Some reports may take a few seconds to load
- Forecasting module is in beta testing
- UI responsiveness on different screen resolutions

Troubleshooting:
---------------
1. If the application won't start:
   - Ensure Java 24 is installed correctly
   - Check that the database is running (green dot in taskbar)
   
2. If you can't connect to the database:
   - Close the application
   - Run stop_db.bat
   - Wait 30 seconds
   - Run start_db.bat
   - Try the application again
